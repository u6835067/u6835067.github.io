About This Site

Aurora Notes is a simple yet modern HTML website designed to show how structure, design, and creativity can work together. It uses only HTML and CSS—no frameworks, no libraries, just clean fundamentals.

"Small ideas, written down, can grow into big things."

Core Features

Responsive Layout

Looks good on desktops, tablets, and mobile devices without extra code.

Modern Design

Uses gradients, cards, and soft shadows for a clean, professional feel.

Beginner Friendly

Easy to read and modify—perfect for learning or school projects.

Single File

Everything is inside one HTML file, so it works great with GitHub Pages.

Daily Thought

Learning web development isn’t about memorizing everything—it’s about understanding how pieces fit together. Start small, experiment often, and don’t be afraid to break things.
